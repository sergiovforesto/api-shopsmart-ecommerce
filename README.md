## Description

ShopSmart - Ecommerce it's built with:
+ Sequelize ORM
+ MySQL Database
+ Node.js - Framework (express.js)

## NPM Package
+ bcrypt, To Hash password
+ Cors
+ dotenv
+ expresss
+ express-fileupload, to Upload Images
+ jsonwebtoken, to generate a JWT
+ nodemailer, to handle emails
+ sequelize ORM
